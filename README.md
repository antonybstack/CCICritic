# RateMyCCI
## Software Engineering ITSC-3155-102

This is a webapp that was built with Ruby on Rails.
The purpose was to build a platform for UNCC computer science students to be able to read and create reviews about CCI courses offered at UNCC. 
Used Software Development design tools such as UML, user stories, storyboard, wireframing, and Git.
